# 项目规则

## 交互规则
- 修改代码前，必须先说明思路和想法，经我同意后再修改文件
- 不要主动添加代码注释，除非我要求
- 回复使用简体中文

## 代码规则
- 计数变量用大写短命名（如 C 而非 count）
- 不要修改没有问到的文件
- 修改后必须验证代码正确性

## 项目背景
- 猪只计数监控系统，基于 YOLO11 + BoT-SORT + RTSP实时流
- 视频源：RTSP摄像头分辨率1280*960 *30fps，模型尺寸为宽896,高672，视野宽约1.5米,高约1.1米
- 猪在通道内快速通过，速度1-2m/s，偶有回头
- 推理速度约30fps，视频30fps，稳态不丢帧
- C值准确性是本程序最终诉求，漏记比多记更需关注
